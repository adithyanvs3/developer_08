# Developer Landing Page 

[live@](https://developer-08.vercel.app/)

A modern and responsive **developer-focused landing page** built using **HTML** and **Tailwind CSS**.  
This project showcases a clean UI layout with API categories, blog sections, call-to-action components, and a structured footer.



---

##  Features

- Fully responsive design (mobile & desktop)
- Modern navigation bar with icons
- API category cards section
- Latest blog articles layout
- Call-to-action sections (Contact & Download)
- Clean footer with social media icons
- Utility-first styling using Tailwind CSS

---

##  Tech Stack

- **HTML5**
- **Tailwind CSS (CDN)**
- **Font Awesome Icons**
